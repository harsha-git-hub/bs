Bootstrap is 1st framework of html,css,javascript.it is designed for responsive webdesigning

	Bootstrap is based on grid system
		About grid system	
			-> The grid system has divided all the devices into 4 types
				1.xtra small
				2.small device
				3.medium device
				4.large device
-----------------------------------------------------------
			Range of each device
				1.xtra small (1px to 777px)
				2.small device (778px to 990px)
				3.medium device (991px to 1200px)
				4.large device ( >1201px )
------------------------------------------------------------
	Grid system has given 12 classes for each device
		Grid system allow to use any 12 class at a time.

		while creating a page using bootstrap follow the steps
			1.Link the bootstrap.css in head section
			2.Follow body tag
			3.Inside body follow the below classes
				a.container (it has some gap from left & right)
				b.container-fluid (it has no gap,it occupies 100%)
			4.Inside container of container-fluid follow the class as
				a.row
			NOTE : height is auto,width is 100% for row class
			5.Inside row follow columns (either of 12 classes from given 48)
